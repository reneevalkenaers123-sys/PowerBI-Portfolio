# Executive Sales Dashboard V3

Interactive Power BI dashboard focused on sales performance analysis, KPI monitoring and filtering insights.

## Overview

This dashboard was built using Power BI and CSV sales data to practice business intelligence reporting, data storytelling and dashboard design.

## Features

- Total Sales KPI
- Average Sales KPI
- Sales by Category
- Sales by Product
- Sales by Region
- Interactive Region Filter
- Interactive Category Filter
- Clean executive dashboard layout

## Business Insights

- Sports category generates the highest sales
- Bike is the top selling product
- Sales are evenly distributed across regions
- Dashboard filters allow quick segmentation analysis

## Tools Used

- Power BI
- CSV Dataset
- Data Visualization
- Dashboard Design
- KPI Reporting
